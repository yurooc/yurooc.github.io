1.1Pandas数据读取#

Pandas需要先读取表格类型的数据，然后进行分析

数据类型 说明 Pandas读取方法
csv、tsv、txt 用逗号分隔、tab分割的纯文本文件 pd.read_csv()
excel 微软xls或xlsx文件 pd.read_excel()
mysql 关系型数据库表 pd.read_sql()

1.1.1示例（读取csv文件，使用默认的标题行、逗号分隔符）：

file_path = '文件路径'
file_ = pa.read_csv(file_path)	#使用pd.read_csv读取数据
file_.head()	#查看前几行数据
file_.shape		#查看数据的形状，返回(行数、列数)
file_.columns	#查看文件里的列名字段名列表
file_.index		#查看文件数据索引列(有多少索引)
file_.dtypes	#查看每列的数据类型

1.1.2示例（读取txt文件，自己指定分隔符、列名）（txt文件格式不同，所以需要自己指定格式）：

path = "路径"
file = pd.read_csv(
	path,
    sep = '\t',		#数据直接用什么分割，\t代表换号符分割
    header = None， #header代表标题行，这里None代表没有标题行，如果有标题列就进行设置
    names =['pdate','pv','uv']	#如果数据没有列名可以自己指定列名，这里用三个字段指定了三个列名
)

1.1.3示例（读取excel文件）：

path = '文件路径'
file = pd.read_excel(path)

1.1.4示例（读取mysql数据表）：

import pymysql
